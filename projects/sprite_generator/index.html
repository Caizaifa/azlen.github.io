Sprite Generator

December, 2015

Video game sprite generator based on Dave Bollinger's Pixel Spaceships.

Features

  • Draw template for sprite generation
  • HSL color range picker
  • Frame Animation
  • X/Y mirroring
Example of animation, spaceships
