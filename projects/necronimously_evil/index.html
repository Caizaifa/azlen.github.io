Necronimously Evil

December, 2012

Necronimously Evil is the first game I created for Ludum Dare when I was 12 years old. It was inspired by 3D-audio game Papa Sangre and built using now-defunct browser audio APIs.

Made for Ludum Dare 25 (theme: You are the Villain).
